# 开源大梳理 · iot

## 家庭自动化

- [Home Assistant](https://github.com/home-assistant/core) — 开源家庭自动化平台。
- [openHAB](https://github.com/openhab/openhab-core) — 开源家庭自动化运行时。
- [Node-RED](https://github.com/node-red/node-red) — 基于流的低代码自动化编排。

## 固件与 MCU

- [ESPHome](https://github.com/esphome/esphome) — 用 YAML 配置 ESP 设备固件。
- [ESP-IDF](https://github.com/espressif/esp-idf) — 乐鑫官方 IoT 开发框架。
- [Tasmota](https://github.com/arendst/Tasmota) — 替代固件，常用于智能插座/继电器。
- [Arduino Core](https://github.com/arduino/ArduinoCore-avr) — Arduino AVR 核心（入门嵌入式生态代表）。
- [PlatformIO](https://github.com/platformio/platformio-core) — 嵌入式开发的专业 IDE/构建系统。

## RTOS

- [Zephyr](https://github.com/zephyrproject-rtos/zephyr) — 可扩展实时操作系统，面向受限设备。
- [FreeRTOS](https://github.com/FreeRTOS/FreeRTOS) — 广泛使用的实时操作系统内核。

## 无线与桥接

- [Zigbee2MQTT](https://github.com/Koenkk/zigbee2mqtt) — 把 Zigbee 设备桥到 MQTT。
- [Z-Wave JS](https://github.com/zwave-js/node-zwave-js) — 现代 Z-Wave 驱动栈。
- [connectedhomeip (Matter)](https://github.com/project-chip/connectedhomeip) — Matter 智能家居互联标准参考实现。
- [Kismet](https://github.com/kismetwireless/kismet) — 无线网络侦测与监测。

## MQTT

- [Eclipse Mosquitto](https://github.com/eclipse/mosquitto) — 轻量 MQTT broker。
- [EMQX](https://github.com/emqx/emqx) — 大规模可扩展 MQTT 平台。

## 边缘 ML

- [TensorFlow Lite for Microcontrollers](https://github.com/tensorflow/tflite-micro) — 微控制器上的轻量推理。
- [Edge Impulse CLI / tooling refs](https://github.com/edgeimpulse/edge-impulse-cli) — 边缘 ML 数据采集与部署工具链入口。
