# 开源大梳理 · finance

## 交易与回测

- [CCXT](https://github.com/ccxt/ccxt) — 加密货币交易所统一 API 库。
- [Freqtrade](https://github.com/freqtrade/freqtrade) — 开源加密货币交易机器人。
- [Zipline-Reloaded](https://github.com/stefan-jansen/zipline-reloaded) — 算法交易回测库（Zipline 社区维护版）。
- [backtrader](https://github.com/mementum/backtrader) — Python 回测与交易框架。
- [vectorbt](https://github.com/polakowo/vectorbt) — 基于 NumPy/Pandas 的高速向量化回测。
- [QuantConnect Lean](https://github.com/QuantConnect/Lean) — 开源算法交易引擎。

## 指标与行情

- [TA-Lib](https://github.com/TA-Lib/ta-lib) — 技术分析指标库。
- [Pandas TA](https://github.com/twopirllc/pandas-ta) — 基于 pandas 的技术分析扩展。
- [yfinance](https://github.com/ranaroussi/yfinance) — Yahoo Finance 行情下载器。
- [AKShare](https://github.com/akfamily/akshare) — 中国财经数据接口库。
- [OpenBB](https://github.com/OpenBB-finance/OpenBB) — 开源投资研究平台。

## 个人财务

- [Firefly III](https://github.com/firefly-iii/firefly-iii) — 自托管个人财务管家。
- [Actual Budget](https://github.com/actualbudget/actual) — 本地优先的个人预算工具。
- [GnuCash](https://github.com/Gnucash/gnucash) — 开源复式记账财务软件。
- [Ledger](https://github.com/ledger/ledger) — 命令行复式记账系统。
- [hledger](https://github.com/simonmichael/hledger) — 易用的纯文本会计工具（Haskell）。

## 计费支付

- [Kill Bill](https://github.com/killbill/killbill) — 开源订阅计费与支付编排平台。
- [stripe-mock](https://github.com/stripe/stripe-mock) — Stripe API 本地模拟服务。
