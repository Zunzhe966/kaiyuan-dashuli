# 开源大梳理 · cms-docs

## 经典 / 出版 CMS

- [WordPress](https://github.com/WordPress/WordPress) — 全球最流行的 PHP 内容管理系统。
- [Ghost](https://github.com/TryGhost/Ghost) — 面向出版的现代 Node CMS。

## 无头 CMS

- [Strapi](https://github.com/strapi/strapi) — 开源无头 CMS，可自托管。
- [Directus](https://github.com/directus/directus) — 把 SQL 数据库瞬间变成无头 CMS/API。
- [Payload](https://github.com/payloadcms/payload) — TypeScript 无头 CMS，代码优先。
- [Keystone](https://github.com/keystonejs/keystone) — 基于 GraphQL 的 Node CMS 框架。

## 文档站

- [Docusaurus](https://github.com/facebook/docusaurus) — React 文档网站生成器。
- [VitePress](https://github.com/vuejs/vitepress) — Vite + Vue 的静态文档站。
- [MkDocs](https://github.com/mkdocs/mkdocs) — Python Markdown 文档静态站点。
- [Material for MkDocs](https://github.com/squidfunk/mkdocs-material) — MkDocs 最流行的 Material 主题。
- [Sphinx](https://github.com/sphinx-doc/sphinx) — Python 生态经典文档工具。
- [docsify](https://github.com/docsifyjs/docsify) — 无需构建的文档站点生成器。
- [Nextra](https://github.com/shuding/nextra) — 基于 Next.js 的文档/站点框架。
- [Starlight](https://github.com/withastro/starlight) — Astro 官方文档主题套件。

## 静态站 / 内容站

- [Hugo](https://github.com/gohugoio/hugo) — 极速 Go 静态站点生成器。
- [Jekyll](https://github.com/jekyll/jekyll) — GitHub Pages 默认静态站引擎。
- [GitBook (legacy open)](https://github.com/GitbookIO/gitbook) — 早期开源 GitBook（现产品已商业化，仓供参考）。


> 注：`astro` 节点在 `web-frontend` 域，与 `starlight` 同生态。
