# 开源大梳理 · mobile

## 跨平台

- [Flutter](https://github.com/flutter/flutter) — 跨平台 UI 工具包，一套代码多端。
- [React Native](https://github.com/facebook/react-native) — 用 React 写原生移动应用。
- [Expo](https://github.com/expo/expo) — React Native 工具链与运行时，加速交付。
- [Ionic Framework](https://github.com/ionic-team/ionic-framework) — Web 技术做移动/桌面的 UI 框架。
- [Capacitor](https://github.com/ionic-team/capacitor) — 把 Web 应用打包到原生容器。
- [.NET MAUI](https://github.com/dotnet/maui) — .NET 跨平台应用 UI 框架。

## Android

- [Android Kotlin guides repo](https://github.com/android/kotlin-guides) — Android Kotlin 官方示例与指南入口。
- [OkHttp](https://github.com/square/okhttp) — Android/Java HTTP 客户端。
- [Retrofit](https://github.com/square/retrofit) — Android 类型安全 HTTP API 客户端。
- [Ktor](https://github.com/ktorio/ktor) — Kotlin 异步客户端/服务端框架，常用于移动网络。
- [Jetpack Compose](https://github.com/androidx/androidx) — Android 现代声明式 UI（AndroidX 单仓入口）。

## iOS

- [Swift](https://github.com/swiftlang/swift) — Apple 平台主语言。
- [Alamofire](https://github.com/Alamofire/Alamofire) — Swift HTTP 网络库。
- [SwiftUI Lab examples mirror note](https://github.com/twostraws/SwiftUI) — SwiftUI 学习与示例（社区高质量入口）。

## 移动测试

- [Detox](https://github.com/wix/Detox) — React Native 灰盒 E2E 测试。
- [Appium](https://github.com/appium/appium) — 跨端移动自动化测试框架。

## 发布与测试增强

- [Maestro](https://github.com/mobile-dev-inc/maestro) — 声明式移动端 UI 自动化测试。
- [fastlane](https://github.com/fastlane/fastlane) — iOS/Android 构建发布自动化。
- [Shorebird](https://github.com/shorebirdtech/shorebird) — Flutter 代码推送更新（Code Push）。
- [Kotlin Multiplatform](https://github.com/JetBrains/kotlin) — Kotlin 多平台共享业务逻辑（含移动）。
