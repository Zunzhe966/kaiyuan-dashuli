# 开源大梳理 · media

## 处理工具链

- [FFmpeg](https://github.com/FFmpeg/FFmpeg) — 音视频录制、转换与流处理的事实标准工具集。
- [GStreamer](https://github.com/GStreamer/gstreamer) — 管道式多媒体框架。
- [HandBrake](https://github.com/HandBrake/HandBrake) — 开源视频转码器（含 GUI）。
- [ImageMagick](https://github.com/ImageMagick/ImageMagick) — 命令行图像处理套件。

## 播放器

- [mpv](https://github.com/mpv-player/mpv) — 基于 mplayer/mplayer2 的自由媒体播放器。
- [VLC](https://github.com/videolan/vlc) — 跨平台多媒体播放器与框架。

## 录播与下载

- [OBS Studio](https://github.com/obsproject/obs-studio) — 免费开源的直播与录屏软件。
- [yt-dlp](https://github.com/yt-dlp/yt-dlp) — 功能丰富的视频下载器（youtube-dl 后继）。

## 媒体服务

- [Jellyfin](https://github.com/jellyfin/jellyfin) — 免费开源的媒体服务器系统。
- [Navidrome](https://github.com/navidrome/navidrome) — 兼容 Subsonic 的音乐流媒体服务器。

## 编辑与创作

- [Shotcut](https://github.com/mltframework/shotcut) — 跨平台开源视频编辑器。
- [Kdenlive](https://github.com/KDE/kdenlive) — KDE 开源视频编辑器。
- [Audacity](https://github.com/audacity/audacity) — 开源音频编辑器。
- [Ardour](https://github.com/Ardour/ardour) — 专业开源数字音频工作站。
- [Blender](https://github.com/blender/blender) — 3D 创作套件，含建模/动画/视频剪辑。

## 视觉与语音

- [OpenCV](https://github.com/opencv/opencv) — 计算机视觉库。
- [whisper.cpp](https://github.com/ggerganov/whisper.cpp) — OpenAI Whisper 的 C/C++ 移植，本地语音转写。
- [Piper](https://github.com/rhasspy/piper) — 快速本地神经网络文本转语音。
