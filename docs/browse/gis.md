# 开源大梳理 · gis

## 桌面与处理

- [QGIS](https://github.com/qgis/QGIS) — 开源桌面 GIS 应用。
- [GRASS GIS](https://github.com/OSGeo/grass) — 强大的地理信息系统处理套件。
- [GDAL](https://github.com/OSGeo/gdal) — 栅格/矢量地理数据转换库。
- [PROJ](https://github.com/OSGeo/PROJ) — 坐标转换与投影库。
- [PDAL](https://github.com/PDAL/PDAL) — 点云数据翻译与处理库。

## 空间数据

- [PostGIS](https://github.com/postgis/postgis) — PostgreSQL 的空间数据库扩展。
- [GeoPandas](https://github.com/geopandas/geopandas) — pandas 风格的矢量空间数据分析。
- [rasterio](https://github.com/rasterio/rasterio) — Python 栅格 I/O（GDAL 友好封装）。

## Web 地图

- [Leaflet](https://github.com/Leaflet/Leaflet) — 轻量移动友好的网页地图库。
- [OpenLayers](https://github.com/openlayers/openlayers) — 功能全面的网页地图库。
- [MapLibre GL](https://github.com/maplibre/maplibre-gl-js) — 开源矢量瓦片地图渲染（Mapbox GL 开源分支）。
- [deck.gl](https://github.com/visgl/deck.gl) — 大规模 WebGL 地理可视化框架。
- [Turf.js](https://github.com/Turfjs/turf) — 浏览器/Node 的空间分析库。
- [Kepler.gl](https://github.com/keplergl/kepler.gl) — 大规模地理空间数据可视化应用。

## 服务与瓦片

- [GeoServer](https://github.com/geoserver/geoserver) — 开源地理数据服务器（OGC 服务）。
- [MapServer](https://github.com/MapServer/MapServer) — 开源地图服务器。
- [tippecanoe](https://github.com/felt/tippecanoe) — GeoJSON 到矢量瓦片的构建工具。

## OSM 分析

- [OSMnx](https://github.com/gboeing/osmnx) — 从 OpenStreetMap 下载/分析街道网络。
