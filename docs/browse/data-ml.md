# 开源大梳理 · data-ml

## DataFrame

- [pandas](https://github.com/pandas-dev/pandas) — Python 表格数据分析库。
- [Polars](https://github.com/pola-rs/polars) — 高性能 DataFrame 库（Rust 核心）。

## 数值计算

- [NumPy](https://github.com/numpy/numpy) — Python 数值计算基础库。
- [SciPy](https://github.com/scipy/scipy) — 科学计算算法库。
- [JAX](https://github.com/jax-ml/jax) — 可组合的数值计算与自动微分。

## 经典机器学习

- [scikit-learn](https://github.com/scikit-learn/scikit-learn) — 经典机器学习工具包。
- [XGBoost](https://github.com/dmlc/xgboost) — 梯度提升树库。
- [LightGBM](https://github.com/microsoft/LightGBM) — 高效梯度提升框架。

## 深度学习

- [PyTorch](https://github.com/pytorch/pytorch) — 主流深度学习框架。
- [TensorFlow](https://github.com/tensorflow/tensorflow) — 端到端机器学习平台。
- [JAX](https://github.com/jax-ml/jax) — 可组合的数值计算与自动微分。

## 大数据

- [Apache Spark](https://github.com/apache/spark) — 大规模分布式数据处理引擎。

## 分析工程

- [dbt Core](https://github.com/dbt-labs/dbt-core) — 分析工程中的 SQL 变换框架。

## 编排

- [Apache Airflow](https://github.com/apache/airflow) — 工作流编排平台。
- [Prefect](https://github.com/PrefectHQ/prefect) — 现代 Python 工作流编排。

## MLOps

- [MLflow](https://github.com/mlflow/mlflow) — 实验跟踪与模型管理。
- [Feast](https://github.com/feast-dev/feast) — 特征存储。

## 可视化

- [Matplotlib](https://github.com/matplotlib/matplotlib) — Python 绑图基础库。
- [Plotly](https://github.com/plotly/plotly.py) — 交互可视化库。
