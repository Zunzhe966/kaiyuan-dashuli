# 开源大梳理 · gamedev

## 引擎与框架

- [Godot Engine](https://github.com/godotengine/godot) — 功能完整的开源游戏引擎。
- [Bevy](https://github.com/bevyengine/bevy) — 数据驱动的 Rust 游戏引擎。
- [MonoGame](https://github.com/MonoGame/MonoGame) — 跨平台游戏框架（XNA 精神后继）。
- [Phaser](https://github.com/phaserjs/phaser) — 快速的 HTML5 游戏框架。

## 底层库

- [raylib](https://github.com/raysan5/raylib) — 简单易学的游戏编程库。
- [SDL](https://github.com/libsdl-org/SDL) — 跨平台底层多媒体/窗口/输入库。
- [SFML](https://github.com/SFML/SFML) — 简单快速的多媒体库。
- [wgpu](https://github.com/gfx-rs/wgpu) — 基于 WebGPU 的跨平台图形抽象。

## Web 3D

- [three.js](https://github.com/mrdoob/three.js) — WebGL 3D 库。
- [Babylon.js](https://github.com/BabylonJS/Babylon.js) — 强大的 Web 3D 引擎。

## 内容工具

- [Tiled](https://github.com/mapeditor/tiled) — 灵活的 2D 关卡编辑器。
- [Aseprite](https://github.com/aseprite/aseprite) — 像素画与像素动画工具（源码可用，二进制授权另议）。
- [Spine Runtimes](https://github.com/EsotericSoftware/spine-runtimes) — 2D 骨骼动画运行时（编辑器商业）。

## 物理/音频/专项

- [Box2D](https://github.com/erincatto/box2d) — 2D 物理引擎。
- [Bullet Physics](https://github.com/bulletphysics/bullet3) — 3D 物理引擎。
- [SoLoud](https://github.com/jarikomppa/soloud) — 易用的跨平台游戏音频引擎。
- [hecs](https://github.com/Ralith/hecs) — 轻量 Rust ECS 库。
- [OpenMW](https://github.com/OpenMW/openmw) — 《上古卷轴 III》引擎重实现。
