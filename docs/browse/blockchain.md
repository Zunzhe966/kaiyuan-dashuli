# 开源大梳理 · 区块链与 Web3

## 以太坊与 EVM

- [go-ethereum (geth)](https://github.com/ethereum/go-ethereum) — Go 实现的以太坊执行客户端。
- [Prysm](https://github.com/prysmaticlabs/prysm) — 以太坊共识客户端。
- [Solidity](https://github.com/ethereum/solidity) — 以太坊智能合约语言与编译器。
- [OpenZeppelin Contracts](https://github.com/OpenZeppelin/openzeppelin-contracts) — 审计友好的代币、权限等合约组件。
- [Foundry](https://github.com/foundry-rs/foundry) — Solidity 合约的测试、脚本与本地开发工具链。
- [Hardhat](https://github.com/NomicFoundation/hardhat) — JavaScript/TypeScript 的以太坊开发环境。
- [ethers.js](https://github.com/ethers-io/ethers.js) — 与以太坊交互的 JavaScript 库。
- [viem](https://github.com/wevm/viem) — 类型友好的现代 EVM TypeScript 客户端库。
- [wagmi](https://github.com/wevm/wagmi) — React 钱包连接与链上读写 Hooks。

## 比特币

- [Bitcoin Core](https://github.com/bitcoin/bitcoin) — 比特币参考客户端。
- [LND](https://github.com/lightningnetwork/lnd) — 比特币闪电网络节点与支付通道实现。

## 其他链与框架

- [Solana](https://github.com/solana-labs/solana) — 高性能 L1 区块链的参考实现。
- [Anchor](https://github.com/coral-xyz/anchor) — Solana 程序开发框架。
- [Substrate](https://github.com/paritytech/polkadot-sdk) — 可定制区块链和 Polkadot 生态框架。
- [Cosmos SDK](https://github.com/cosmos/cosmos-sdk) — 构建应用特定区块链的框架。

## Web3 基础设施

- [IPFS Kubo](https://github.com/ipfs/kubo) — 内容寻址的分布式文件存储实现。
- [libp2p](https://github.com/libp2p/go-libp2p) — 模块化点对点网络栈。
- [The Graph](https://github.com/graphprotocol/graph-node) — 区块链事件索引协议实现。
