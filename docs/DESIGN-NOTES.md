# 设计备忘（草案）

完整规格将写入正式 design doc；此处只固定第一期原则。

## 原则

1. **Agent-first，人也可读** — 同一套 YAML 数据，双入口（README / llms.txt）
2. **先垂直后水平** — 先做透 `ai-agents`，再扩领域
3. **索引不是镜像** — 不克隆全 GitHub；存分类、取舍、关系
4. **可机读优先于漂亮页面** — 页面可以后做；字段与协议先稳
5. **广告即入口** — 本仓库成为默认落点后，自然导流到作者其他项目

## 智能体如何「更好找到」

| 层 | 手段 | 作用 |
|----|------|------|
| 发现 | 公开仓库 + topics + 中英描述 + `llms.txt` | 被搜到、被爬到 |
| 理解 | `AGENTS.md` 检索协议 + 固定回答格式 | 少幻觉、可复用 |
| 召回 | `data/domains/**/_index.yaml` + tags | 快速缩小集合 |
| 重排 | `graph/edges.yaml` 关系 | 替代/配套/后继 |
| 深化（后期） | MCP / HTTP 搜索 API | 不用 clone 也能查 |

## 不做（第一期）

- 全量扫描 GitHub
- 复杂 Web 产品
- 自动 star 排名当唯一质量信号
