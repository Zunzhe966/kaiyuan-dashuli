核查日期：2026-07-19。

## 公开范围

- 公开总仓库：`https://github.com/Zunzhe966/kai-yuan-da-shu-li`。
- 当前本机构建基线：19 个领域、389 个项目、406 条关系；生产站以其自身 `api/v1/meta.json` 为准。
- 本分支不复制主工作区另一个窗口尚未提交的数据；后续只通过正式 Git 合并接收它们。
- 当前不创建空的分类仓库。未来只有领域需要独立维护、单仓库构建明显变慢或仓库接近 GitHub 推荐的 10GB 时才拆分。

## 私人范围

- 私人仓库名称、用途、本机路径、凭据和开发状态不进入本公开仓库。
- 私人项目清单只保存在私人开发同步仓库或本机管理目录。
- 公开网站只展示已经公开的开源目录数据，不展示中转站账号、代理、服务器密钥或个人资料。

## 搬瓦工只读核查

2026-07-19 通过 SSH 只读检查确认：

- 没有名称包含 `kai-yuan`、`atlas` 或 `graph` 的应用服务；
- 没有监听 `8787` 或 `8088` 的端口；
